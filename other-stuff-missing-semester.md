## Git and Github

Git is commonly used to store and track changes to
source code (programming text) repositories and
related documentation.

Github is a popular website based on the git tool that
allows easily publishing and sharing git repositories,
and web-based collaboration features.

Both of these would be useful to a working mathematician,
or any other persion doing technical work with tools that
read or manipulate text files.

[Intro to github and git](https://docs.github.com/en/get-started/start-your-journey/about-github-and-git
)

## Markdown (Github flavor)

Markdown is an open standard (with various variants
or "flavors") for authoring and viewing rich text
documents where the content is also easily readable
and writable in plain text form.

Read the [intro to Github's Markdown flavor](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax)

Visual Studio code also recognizes Markdown files for file
names ending with `.md`. When editing Markdown, a toolbar
button to the right of the open tabs (with a magnifying
glass over a divided page) should allow showing a preview
of the rendered/formatted Markdown. Or `Shift-CMD-V` (Mac)
will show a preview in a new tab.
